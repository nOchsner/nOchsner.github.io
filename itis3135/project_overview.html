Project Overview

  • This Application will help serve as an introduction to JavaScript. It will not only give information on how to utilize JavaScript but also have interactive sections.
  • Intended to not only to teach users and introduction to JavaScript but to also to serve as a place where more experience users can quickly refresh their minds on forgotten topics.
  • This Website will be split into parts describing the different functionalities of the JavaScript Language. Starting with simple variable declarations to interating with the DOM. This will also include practice question where users can test their knowledge.

Page Design

Home

Purpose and Audience: This page is to serve as a home page for the website. It'll tell the users what this website will provide for them and more. This page is meant for users that want to get an overview of what this website provides.

Content and Actions: This page will include a nav bar leading to the other main pages, as respresented in the site map, and will serve as an introduction. There will be no need for users to enter data to this page and no major actions other than switching to the other pages.

Learning/Topics

Purpose and Audience: This is one of the two main sections. This page will also include a second nav-bar of sorts to navigate through the various Topic sections. The Topic pages are more of sub-pages to the main learning page. The main audience for this part is for learning the content rather than being tested on it.

Content and Actions: This will include much interactivity and actions through the use of JavaScript. It'll have practice and learning parts to help with the learning of certain JavaScript topics. These pages will serve mostly as a precursor to the Quizzes pages in which users will be tested on their knowledge. This part will include interactivity to not only allow users to see how their code can affect the DOM but also the structor that JavaScript follows by teaching them how it works.

Quizzes/Topic Quizzes

Purpose and Audience: This is the second of the two main pages. Whether the use of this pages is to test your knowledge on JavaScript or as practice after running through the learning pages this will serve as a way for general users to test their knowledge.

Content and Actions: The content of these Quizzes pages will be to test the users on specific topic. The sub-pages will be used to differentiate between the different Quizzes based on the topic that the user would like to be tested on. This part will include interactivity to not only allow users to see how their code can affect the DOM and also the structure that JavaScript follows by testing them.

About

Purpose and Audience: This page will serve as an about section. It is meant to give more information about me and other related items. This will be for general users and will be rather short.

Content and Actions: This will be a relatively static page without much action. This pages content is really only meant to serve as a source for some general information. There will be multiple sections to it but it will mostly be written information.

Common Questions

Purpose and Audience: Similar to the home page this page will be primarly to inform rather than have any interactivity. This page will include questions that a user of this website may have. For example: "How do you access the Quiz answers?". This would come with a response given in detail on how the users can see the answers to the quizzes to they know what they got wrong and why.

Content and Actions: This page will most likely not have any major actions so there is no need to elaborate on that. As for its content, as mentioned previously, there will be common questions and answers to help users navigate the site.

Dynamic Functionality

  • Will include practice questions within the learning sections where users can answer questions and see the answers.
  • Contains a tests section where users can take practice tests and answer a group of related questions.
  • Specific parts in the learning section where users can see exactly how specific functions work by simulating JavaScript and a console.
  • Some of the dynamic content will be similar to the Free Code Camp activities.
